# Play: Research / synthesis / proposing what to build — mine self-skeptically

> Carried from the original observation; **not yet re-tested by a falsifiable run here.** Ground it in your own work before trusting it.

- **Adversarial pass over your OWN output before delivery.** Kill anything that duplicates something already built, needs a capability you lack, or is scaffolding rather than a mechanism.
- **Demand ≥2 independent sources before elevating an idea, and state the count.** One source is a lead, not a finding.
- **Verify named external entities** (model IDs, package versions, endpoints) resolve before depending on them; if one doesn't, STOP and surface the disproof rather than substituting the nearest plausible match.
- **Keep a visible "examined / refuted — with reason" list** so killed ideas stay dead, and name at least one surviving weakness in what you deliver.
